## 0. 运行时总合同

### 0.1 场景执行约定

- 每个场景是一个可独立进入、保存、恢复和清理特效的运行时单元；连续追逐只在明确的场景切点切换。
- “可发声内容”只允许人物真实对白、电话、录音、语音留言及世界内广播；“静音旁白”承载动作、心理、环境和系统说明。
- 玩家界面不显示本文件中的场景 ID、状态 ID、数值、真假标签或结局判定条件。
- 重要选择至少在后续一个场景再次产生可感知结果；不使用即时“正确/错误”判定卡。
- 手机、规则册、线索册和关系变化均采用非阻断表现。读档、离场、切换故事时必须清理震动、闪烁、失焦、抖动和音频循环。

### 0.3 身份验证合同

1. 旧记忆、账号、学籍、照片和公共口令只能作为一类证据，不能单独判定真人。
2. 当夜未联网记录的新事件、选择原因、临时承诺和自愿承担风险构成第二类证据。
3. 羽毛反应只证明污染路径，不证明道德主体或原身身份。
4. 角色作出的后续选择构成第三类证据，决定其是否拥有“第二人生”的资格。
5. 第三章给出第一次临时判断，第五章用身份消失反驳旧证据，第七章才完成最终判断。

## 第一章《不要忘记你的名字》

- **阶段目标**：让玩家主动处理异常短信、完整阅读八条学生规则，经门外宋明与红马甲追逐验证规则，并在抵达三楼前亲历第一次姓名阻塞。
- **恐怖升级**：私人短信异常 → 班群被接管 → 走廊瘦长影子 → 多方向宋明声线 → 隔壁开门后骤然死寂 → 红马甲套取人数并破门 → 楼梯空间循环与数字 4。
- **不可逆后果**：最初短信是否留存、是否公开宋明全名、是否泄露宿舍人数、隔壁学生死亡、追逐伤势与林峰姓名首次卡顿。
- **关系变化**：周朝阳从分析者变为阻止开门和组织逃生的人；谷雨从冲动回应转为主动替林峰喊名；林峰可因被控制而怀疑周朝阳，也可因获救建立信任。
- **规则认知变化**：规则从“恶作剧文本”变为可验证的生存信息；正式查房时间不代表来者可信；“没有四楼”被数字 4 的实际出现首次击穿。
- **钩子**：林峰在三楼门前说出“我叫林……峰”，半掩的宿管宿舍门随谷雨的喊名自行打开。

### DR-C1-S01 陌生短信

- **场景ID**：`DR-C1-S01`
- **章节**：第一章《不要忘记你的名字》
- **名称**：陌生短信
- **时间**：00:00—00:02
- **地点**：男生宿舍 1107，林峰床位
- **人物**：林峰；周朝阳、谷雨、宋明均在低光或床帘后，仅作环境存在
- **前置状态**：新游戏；`nameStage=1`；四人宿舍关系正常；宋明准备去洗手间。
- **进入原因**：手机在午夜整点震动，陌生号码发来“不要忘记你的名字。”
- **目标**：让玩家第一次主动处置信息，并埋下可跨章验证的原始证据。
- **事件**：宋明披毛巾离开，门锁声被短促震动盖住；短信无署名、无号码归属，林峰觉得像舍友恶作剧。
- **动机**：林峰想维持日常解释；未来的黑色头像林峰试图让过去的自己留下姓名锚点。
- **互动**：可打开短信详情；五选一：拉黑删除、回复“你是谁”、截图保存、转发宿舍群、暂时保留。
- **选择后果**：拉黑使第五章取证更困难但保留时间闭环冲击；回复让黑色头像提前锁定当前账号；截图生成未被消息撤回影响的证据；转发使三名舍友更早参与但让四楼复制公开内容；保留允许第二次震动提醒。所有结果在第五章回收。
- **状态**：写入 `savedFirstSms`、`smsReplied`、`smsShared`；若删除，联系人列表中短暂出现一个空白行作为第一阶段可见污染。
- **线索**：线索“午夜警告”；时间戳固定为 00:00；短信措辞与第八结局完全一致。
- **因果连接**：宋明离门与手机震动同时发生，导致林峰没有确认门是否关严；进入 `DR-C1-S02`。
- **人物图**：林峰手持手机的胸像或手部近景；宋明背对镜头披毛巾的低光半身，不展示异常。
- **背景**：9:16 男生宿舍 1107 夜景，四张床与门口同画面，门缝留一线走廊冷光。
- **手机画面**：锁屏 00:00、陌生短信、五个处理入口；删除后不弹正确性提示。
- **声音/特效**：单次手机震动、远处水管声、门轴轻响；选择删除时仅做轻微画面失焦。
- **可发声内容**：宋明：“我去趟厕所。”；林峰低声：“谁这么无聊？”
- **静音旁白**：“午夜十二点，整栋楼像突然把呼吸压低了。”；短信正文作为手机文字静音呈现。

### DR-C1-S02 黑色头像

- **场景ID**：`DR-C1-S02`
- **章节**：第一章《不要忘记你的名字》
- **名称**：黑色头像
- **时间**：00:02—00:06
- **地点**：1107，林峰床位与班群界面
- **人物**：林峰；黑色头像（仅账号）；熟睡的周朝阳、谷雨
- **前置状态**：继承短信处置结果；宋明已离开；宿舍门未被确认关闭。
- **进入原因**：平时活跃的班群突然全体禁言，黑色头像发布规则。
- **目标**：强制完成初始规则的一次性完整阅读，并让玩家发现人数矛盾。
- **事件**：群聊顶部人数先闪烁后恢复原数；黑色头像资料无法打开；《宿舍文明守则》八条在同一条长消息中完整出现。
- **动机**：黑色头像用被占据的林峰群位发送警告；无名层也借同一文本引导学生走向宿管宿舍。
- **互动**：玩家必须滚动读完八条；可长按逐条标记；可查看群成员、点击黑色头像、截屏、尝试发言。
- **选择后果**：查看成员可提前发现林峰自己不在列表但画面迅速恢复；截图保留完整规则；只快速滑到底会让后续回忆选项缺少具体条文；尝试发言显示“你不是本群成员”，随后被系统自动抹去。
- **状态**：解锁规则册；八条规则初始标记为“暂时可信”；若发现群位异常，写入 `clueGroupCount=true` 并推进可见污染但不直接增加阶段。
- **线索**：完整八条《宿舍文明守则》；“黑色头像占位”“群人数未增加”“资料不可查看”。
- **因果连接**：规则第 4 条与未关的门直接冲突；林峰注意到门缝，进入 `DR-C1-S03`。
- **人物图**：林峰眼部与手机反光近景；不为黑色头像绘制实体人物图。
- **背景**：1107 同景，环境更暗，手机是主要光源。
- **手机画面**：一次性完整显示八条规则；群成员总数不增加；黑色头像为纯黑圆形且资料页为空。
- **声音/特效**：连续低频通知震动后突然静音；点击头像时短促数字噪点；规则文字不得闪烁到无法阅读。
- **可发声内容**：林峰：“多了个头像……人数怎么没变？”
- **静音旁白**：八条规则全文以世界内消息静音呈现；“禁言标记旁边，林峰自己的头像像从未存在过。”

### DR-C1-S03 门缝里的影子

- **场景ID**：`DR-C1-S03`
- **章节**：第一章《不要忘记你的名字》
- **名称**：门缝里的影子
- **时间**：00:06—00:09
- **地点**：1107 宿舍门内
- **人物**：林峰、周朝阳、谷雨；走廊瘦长影子
- **前置状态**：八条规则已读；门仍虚掩；宋明未归。
- **进入原因**：林峰意识到规则要求“不要给任何人开门”，而宿舍门本就没有关紧。
- **目标**：完成关门动作并唤醒两名队友，建立共同决策环境。
- **事件**：林峰下床推门；门缝外一道人形影子被走廊灯拉得过分瘦长，脚步却没有声音；门合拢前影子突然停住。
- **动机**：林峰想补救疏忽；影子在确认宿舍内人数和可被叫出的姓名。
- **互动**：可直接关门、先向外看、用手机拍门缝、叫醒周朝阳、叫醒谷雨；只能完成两项后影子逼近。
- **选择后果**：先关门保持信息最少但减少暴露；拍摄得到只有走廊没有影子的照片，增强周朝阳信任；喊队友名字会让门外第一次学到对应声线；只叫谷雨会导致他惊醒时险些大声喊宋明。
- **状态**：房门锁定；更新 `zhouTrust` 或 `guTrust`；记录门外已获知的姓名集合；解锁线索“影像与肉眼不一致”。
- **线索**：影子异常瘦长、无脚步声、手机拍不到；宋明离开时未关门。
- **因果连接**：谷雨核对群人数，周朝阳得出“多了一个人，人数却没增加”；三次敲门触发 `DR-C1-S04`。
- **人物图**：林峰扶门手部特写；周朝阳警觉坐起胸像；谷雨从被子里探头的半身；门缝影子只作背景轮廓。
- **背景**：1107 门内视角，门缝冷光切割前景。
- **手机画面**：若拍摄，显示无影空走廊；群人数核对页可随时回看。
- **声音/特效**：床架轻响、门轴摩擦、锁舌重扣；影子停住时环境声抽空半秒。
- **可发声内容**：谷雨：“宋明还在外面。”；周朝阳：“多了一个人，人数却没有增加。”
- **静音旁白**：“影子没有动，长度却越过门槛，先一步伸进了宿舍。”

### DR-C1-S04 门外宋明

- **场景ID**：`DR-C1-S04`
- **章节**：第一章《不要忘记你的名字》
- **名称**：门外宋明
- **时间**：00:09—00:20
- **地点**：1107 宿舍门内外
- **人物**：林峰、周朝阳、谷雨；门外宋明声源
- **前置状态**：三人清醒；房门锁定；门外至少掌握部分姓名。
- **进入原因**：门外三次规律敲门，宋明声称忘带门卡。
- **目标**：让玩家进行第一次多证据验真，并决定向模仿者泄露多少信息。
- **事件**：周朝阳阻止直接开门；门外声音从正常宋明逐渐变尖，并从门、窗、床底三个方向同时出现；十分钟后突然停止。
- **动机**：谷雨想救舍友；周朝阳要控制信息；门外东西需要完整姓名、人数或回应建立复制。

- **阶段目标**：查明学校如何用固定人数名册维持现实，迫使玩家在恢复自己、同伴、被牺牲者与有自我意识复制体之间作不可逆取舍。
- **恐怖升级**：失踪档案自行更新 → 历届照片人数恒定 → 名册主动删名补位 → 候选者在门外同时求救 → 写回一人时另一人当场被世界遗忘 → 名册深柜放出拥有完整身份的另一个林峰。
- **不可逆后果**：恢复与被挤出的每个姓名、名册是否损毁、吴阿姨或林峰是否被删除、被困学生与复制体能否获得现实身份。

- **钩子**：名册完成最后一次改写后，校长室深柜打开，一个拥有完整学籍、账号、家庭信息和同学认知的林峰走出来。

## 第七章《谁是林峰》

- **阶段目标**：让两个林峰在旧记录、新记忆和承担风险三类证据下完成最终验证，并由累计选择自然落入八种情绪方向不同的结局。
- **恐怖升级**：合法身份压倒当夜经历 → 新承诺可能已泄露 → 两个林峰都能提出合理解释 → 出口要求留下一个姓名 → 无名层穿回现实 → 结局才回收谁被记住、谁被替换、谁作出人的选择。
- **不可逆后果**：谁使用林峰之名、谁留在四楼、谁为出口承担姓名代价、哪些人或复制体进入现实、时间闭环是否形成。

- **规则认知变化**：名字和记忆只能证明过去，面对风险的选择证明现在；正确人数、安全身份与人性不是同一件事。
- **钩子**：本章不再设置下一章钩子；三个最终场景完成累计判定，随后进入八个独立结局之一。
